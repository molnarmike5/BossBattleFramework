hier eine kurze README zu meine Abgabe. :)

Ich habe zwar versucht alles so zu designen, sodass es verständlich ist, ich dachte ein README kann trotzdem nicht schaden.

Ich habe jetzt eine Arena mit Wände gemacht und kurz Materials drauf, damit es nicht so blank aussieht.

Ansonsten befinden sich 3 Prefabs in der Scene, davon ist eine schon vorgeneriert.

Das Tool generiert die Animationsgraphen und die Behavior Script von Boss, sodass die zusammenpassen und synchronisiert sind, dazu noch die nötige und erwünschte Komponente,
außerdem habe ich versucht so viel Focus auf wiederverwendbarkeit und optionalität zu legen wie möglich.

Momentan kann man nicht nachträglich einfacher weise Änderungen an den Moveset oder Phase zusammensetzung vornehmen, außer man macht es manuell in Animationsgraph.
Man kann die enablen und disablen, sowohl moves als auch ganze movesets.

Solltest du irgendwo hängen oder nicht weiterkommen, oder stößt du an einem Bug was alles unbenutzbar macht, melde dich bitte, schaue ich mir gerne an.

Die Lösung ist auf keinem Fall perfekt und fertig, es war viel mehr involviert als ich dachte, aber ich finde es ist eine gute Anfang und man kann darauf sehr gut aufbauen. :)
